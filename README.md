## 📌 Barber Club Landing Page

Landing page moderna y de alta conversión para una barbería premium, desarrollada a partir de un diseño en Figma y llevada a producción con un enfoque en maquetación profesional y experiencia de usuario.

🔗 **Demo en vivo:**  
[https://distrito-barber-club-landing.vercel.app/](https://distrito-barber-club-landing.vercel.app/)

🎨 **Diseño en Figma:**  
[https://www.figma.com/make/n1v5cChM8I0OVgne4EAvkQ/High-Conversion-Landing-Page?t=gV2kW8MMe96aF4IW-1](https://www.figma.com/make/n1v5cChM8I0OVgne4EAvkQ/High-Conversion-Landing-Page?t=gV2kW8MMe96aF4IW-1)

---

## 🚀 Características

- Diseño basado de trabajo realizado en Figma 
- Layouts profesionales con **CSS Grid y Flexbox**  
- Experiencia **responsive** en múltiples dispositivos  
- Efectos visuales con `hover`, `transform` y `transition`  
- Navegación fluida entre secciones (scroll)  
- Estructura CSS organizada (metodología tipo **BEM**)  

---

## 🛠️ Tecnologías utilizadas

- HTML5  
- CSS3  
- Google Fonts  
- Vercel (deploy)

---

## 🎯 Objetivo del proyecto

Este proyecto fue desarrollado para simular un caso real de trabajo frontend:

- Aplicar buenas prácticas de maquetación  
- Construir una landing orientada a conversión  
- Crear un proyecto sólido para portafolio
